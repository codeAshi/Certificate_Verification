## ✨ Features

- 🔐 Admin Authentication System (Login)
- 📂 Bulk Student Certificate Upload via Excel
- 🏷 Unique Certificate ID Generation
- 🔎 Instant Certificate Verification by ID
- 📦 MongoDB Database Integration
- 🌐 Responsive React Frontend
